Citibike Dashboard  


The Citibike explorer is a general-purpose visualization dashboard setup to display rides on New York City's public bikeshare system. The visualization shows the rides for July 17, 2014, the busiest day of use to date.



The dashboard is meant to be explored and has three main parts.


1. Brushes and checks that allow you to filter and understand different types of riders and riding behavior.




2.Outputs on top, showing computed values.




3. A map representing the average riding path of each Citibike station in New York City. The paths redraw as filtering occurs.




 The dashboard is made in such a way that you use all three features to understand riding behavior. For example, to select only riders who took southward trips, were between the ages of 15 and 40 and rode for between 0 and 30 minutes, you would use the two radials below and the previously shown radial above. The map will show the average paths for these riders from each Citibike station.




While in its current setup the dashboard displays bike path data, it could easily be used for any path or indicator data mapping where multiple factors contribute to direction, such as wind, traffic or economic or political change maps. Find the project on Github.
Note on data sourcing and analysis: The Citibike dataset is a heavily-tailed distribution. To create the interactive experience, only rides lasting under 1 hour are included. The raw dataset also shows the presence of several riders over 100 years old. Because these tail values are likely erroneous, only riders under 80 years-old are included. A full repo, including IPython notebooks, is available on Github . Citibike dataset available from Citibikenyc.
